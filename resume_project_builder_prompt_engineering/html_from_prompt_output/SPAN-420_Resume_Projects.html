SPAN 420: Spanish‑English Dictionary Tools

  • Interactive CSV Dictionary Search CLI (Python): Console utility that scans a Spanish–English CSV dictionary line‑by‑line and prints a user‑specified number of matches.
  • Sublime Dictionary Language‑Selector Plugin (Python): Sublime Text plugin that lists all installed `.dic` files and lets users switch the active spell‑check dictionary through a quick‑select panel.
  • Cross‑Platform Dictionary Search Application (C++ / Python): C++ and Python console programs for rapid substring search across Spanish‑English dictionaries, supporting Unicode input and configurable result limits.
